1 Features

This template was designed mostly inspired by wikipedia to create readible HTML pages, and enables user to easily edit the content. It is perhaps most useful to create course pages, scientific online articles and documantation. Specifically it has the following features:

  • Allows writing \( \LaTeX \) in web using Mathjax.
  • Provides easy and quick formatting for HTML using Markdown.
  • Highlights source codes in various languages using code2html script.
  • Automatically creates table of contents for the document.
  • Designed for simplicity, readibility and portability.

2 Writing \( \LaTeX \) in HTML

This template is using Mathjax's online javascript [+] to convert LaTeX equations to images.

Here is an example of a formula written in HTML environment: \\( \pi = 4 \sum_{i=0}^{\infty} \frac{(-1)^{i}}{2i+1} \\), which generates: \( \pi = 4 \sum_{i=0}^{\infty} \frac{(-1)^{i}}{2i+1} \). All you need to do is simply put the equations (in LaTeX syntax) between \\( \\) tag, wherever you want the equation to occur.

Here is another example of a HTML code with LaTeX in it:

\end{array} \right) \;\;\;
\int_{-1}^{1} \frac{1}{\sqrt{1-x^{2}}} dx = \pi \;\;\;\;
1 < 2

5 Example of code highlighting in HTML

This template is using code2html perl script [+] to highlight source codes that are being copied to the html file.

All you need to do is simply put the source code between <C> </C> tags (for C code) and similarly for other languages as shown below.

Following source code shows how you can highlight C, C++ or Perl code:

5.1 C


<C>
#include <stdio.h>

int main(void)
{
    /* The hello world code: */
    printf("hello, world\n");

}
</C>

Which will generate a highlighted code as following:


#include <stdio.h>

int main(void)
{
    /* The hello world code: */
    printf("hello, world\n");

}

5.2 Perl

<PERL>
 #!/usr/bin/perl
 use strict;
 use warnings;
 use IO::Handle;

 #Perl comment

 my ( $remaining, $total );

 $remaining = $total = shift(@ARGV);

 STDOUT->autoflush(1);

 while ( $remaining ) {
  printf ( "Remaining %s/%s \r", $remaining--, $total );
  sleep 1;
 }

 print "\n";
</PERL>

Which will transform to:
